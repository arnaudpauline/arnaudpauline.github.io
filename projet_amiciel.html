Projet de stage chez Amiciel


Présentation de l'entreprise


Amiciel est une SARL située dans la zone industrielle de Saint Quentin Fallavier en Isère.
Elle propose des prestations telles que :

  • Installation d'équipement informatique (badgeuses, tablettes, distributeur de plateaux)
  • Conception et vente d'application métier (Malice, Sacha : tous deux des logiciels de gestion)
  • Infogérence
  • Hébergement

Les clients premier de l'entreprise sont des crèches, écoles, centre spécialisé pour les jeunes... Durant toute la durée de mon stage, j'ai été suivie par...


Présentation de la mission


J'ai du faire évoluer un projet déjà existant sur leur application Malice (logiciel de gestion). L'objectif était de suivre la gestion des contrats de maintenance à l'aide d'un module web.
Pour cela il fallait avoir un formulaire afin de renseigner chaque nouveau contrat. Par la suite un système de tri de tous les contrats permet d'en sélectionner certain dans le but de créer les contrats sous forme de Word et pouvoir les éditer (publipostage).




Présentation de la technologie utilisée


Blazor est une technologie assez récente. Elle premet d'utilser du .NET afin de réaliser l'intégralité d'un site web. Le code C# est compilé dans une DLL qui sera exécuter dans le client de la machine.
